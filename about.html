技术细节

展开查看博客的技术实现

纯静态站点，HTML / CSS / JS 直出。文章存放于 posts/*.md，推送后由 GitHub Actions 自动解析 YAML 头生成 posts.json 索引。评论系统使用 Giscus（基于 GitHub Discussions）， 数学公式使用 KaTeX 渲染，访客统计使用不蒜子。

文章 YAML 示例： 

---
title: 我的第一篇博客
date: 2026-04-24
categories:
  - 孤岛
tags:
  - 随笔
pinned: false
excerpt: 摘要
---

分类择一：孤岛、荒原、摆渡、圆舞。未填默认「孤岛」。 支持行内 $...$ 与块级 $$...$$ 数学公式。 Obsidian ![[图片]] 语法会自动转换为标准 Markdown。
